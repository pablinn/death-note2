backup 

syntax: backup [options]

options:

    to = (server ip)[:filename] {stores config file to tftp server}
    from = (server ip):[filename] {restores config file from tftp server}
    {note the default backup filename is config.rci}

    print {prints the config}
    passwords {includes encrypted passwords and keys}
    differential {differential backup relative to device defaults}

For the "to" and "from"options, a local file for (filename) may be
selected by specifying "local" as the (server ip) value.

Example:  backup to=local
    Store config file in the local filesystem, default file config.rci.
Example:  backup to=local:backup.cfg
    Store config file in the local filesystem, file backup.cfg.
Example:  backup to=local:A/
    Store config file on the A/ volume, default file config.rci.
Example:  backup to=local:A/backup.cfg
    Store config file on the A/ volume, file backup.cfg.

Similar commands may be used with "from" to restore the config file.
